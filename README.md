# SolarSmart+

## Project Overview
SolarSmart+ is a landing page for a smart solar solution platform. It helps users understand solar installation by providing insights like estimated cost, savings, and return on investment based on their electricity usage.

The platform simplifies the decision-making process for homeowners and small businesses who want to switch to solar energy.

---

## Technologies Used
- HTML (for structure)
- CSS (for styling and layout)
- JavaScript (for basic interactivity)

---

## Instructions to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html` or open it using any web browser (Chrome recommended)

---

## Features
- Clean and responsive design
- Structured sections (Home, About, Problem, Solution, Contact)
- Interactive button using JavaScript
